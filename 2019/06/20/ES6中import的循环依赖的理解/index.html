a.js

b.js

babel-node a.js -> 执行完毕

  1. import {bar} from ‘./b.js’ -> 此时对b模块只是一个引用, 不会去执行
  2. foo() -> bar() -> 执行b
  3. import {foo} from ‘./a.js’ -> 指向a.js 中的foo对象
